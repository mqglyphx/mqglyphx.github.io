Front End Developer

Location: Pune

What would you be doing?

Primarily, you will build a scalable front end application in clear and beautiful code. You will work closely with the product and back-end engineering to make decisions which resonate with the front end approach, participate in design and code reviews & independently take UI/ UX decisions.

Above all, you will have the opportunity to sit back and enjoy some of the world beating products that you built.

Job Requirements

  • Bachelor's degree or higher in a related discipline
  • Self-Starter who is curios to improve her/his skills
  • Good design sense, aesthetics and an aptitude for usability
  • Have Experience (1+ year) in
    • JavaScript
    • JS libraries like jQuery, Underscore.js, Backbone.js, Angular JS or Ext JS
    • Strong skills in HTML5 and CSS3
    • Knowledge of RESTful services
    • Preferred (not mandatory): Ruby, Python, Database development
    • Building user friendly web 2.0 interfaces
